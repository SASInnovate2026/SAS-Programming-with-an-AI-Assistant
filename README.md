# SAS Programming with an AI Assistant

Explore how to effectively use ChatGPT as a programming partner when writing SAS code. Learn best practices for prompting and validating results to accelerate your programming workflow. See how ChatGPT can assist with drafting, explaining, and refining SAS programs.
